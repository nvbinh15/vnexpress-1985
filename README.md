# VnExpress 1985

A satirical, fictional edition of VnExpress written as if the modern news website existed in **September 1985** — the late subsidy era (thời bao cấp), anchored on the currency reform of 14/9/1985. Same modern red chrome, comment sections, view counts and SEO headlines; the content is ration coupons, rice books, Honda Cubs and the eternal queue. The anachronism of the medium is the joke.

**Live:** https://vnexpress1985.nvbinh.com

## Register

Warm, wry, nostalgic — the "thương nhớ thời bao cấp" voice of Vietnam's nostalgia exhibits, never bitter or political. The hardships (đổi tiền wiping savings, queuing from 4am) are handled with the affection of people who lived them and laugh about them now. All hero images are generated in a black-and-white 1980s documentary-film aesthetic.

## Stack

Vite + React 19 + TypeScript + Tailwind; MDX articles + JSON comments; `gpt-image-2` → AVIF image pipeline (`scripts/generate-images.mjs`). Static SPA on Vercel.

```bash
npm install && npm run dev
```
